
****************************************************************

		ティラノスクリプト用テーマ一括変換プラグイン

****************************************************************

　ダウンロードいただき誠にありがとうございます
　ゲーム制作のお役に立てれば幸いです
　動作確認はティラノスクリプト v475 で行っています

――――――――――――――――――――――――――――――――――――――――――
■機　能
――――――――――――――――――――――――――――――――――――――――――
　このプラグインを使用することでゲームのシステムテーマを簡単に変更することができます。

　変更できるテーマ

　・メッセージ枠
　・機能ボタン
　・メニュー画面
　・セーブ/ロード画面
　・バックログ画面
　・コンフィグ画面
　・glink
　
――――――――――――――――――――――――――――――――――――――――――
■つかいかた
――――――――――――――――――――――――――――――――――――――――――
------------------------------------------------------------------------------------------------
①プラグインの配置
------------------------------------------------------------------------------------------------

　プロジェクトフォルダの data/others/plugin に解凍した theme_kopanda_12 を配置します。

------------------------------------------------------------------------------------------------
②テーマの反映
------------------------------------------------------------------------------------------------

　first.ks の最初のほうに以下のスクリプトを記述すると即時に反映されます。
　
　[plugin name="theme_kopanda_12"]
　
（注）first.ksでプラグインを読み込んだあとに
　[position]などでメッセージ枠の設定を上書きしないようにご注意ください。

------------------------------------------------------------------------------------------------
③システムボタンの表示
------------------------------------------------------------------------------------------------

　テーマを反映させた時点ではシステムボタンが非表示となっています。

　システムボタンを表示させるには[add_theme_button]タグを記述します。
　非表示にしたいときは[cleafix]タグを使用してください。

　システムボタンの位置や不要なボタンなどは
　theme_kopanda_12フォルダ内の init.ks を編集することでカスタマイズが可能です。

------------------------------------------------------------------------------------------------
④引数について
------------------------------------------------------------------------------------------------

　プラグインの導入時に引数を渡すことができます。
　デフォルトで用意しているのは以下の３つです。

　font_color: メッセージの文字色を指定できます。
　name_color:名前欄の色を指定できます。
　frame_opacity:メッセージ枠の透明度を0〜255の範囲で指定して下さい。0が完全に透明

（例）
　[plugin name="theme_kopanda_12" font_color="0xFFFFFF" name_color="0xFFCC00"  frame_opacity="255" ]

------------------------------------------------------------------------------------------------
⑤glinkの使用
------------------------------------------------------------------------------------------------
　テーマに合ったglinkも自動的に使用できるようになります。
　color に ts11 を指定して下さい。

　[glink  color="ts12"  storage="scene1.ks"  size="20"  x="260"  width="400"  y="100"  text="はい。興味あります" target="*selectinterest"  ]
　[glink  color="ts12"  storage="scene1.ks"  size="20"  x="260"  width="400"  y="170"  text="興味あります！"  target="*selectinterest"  ]
　[glink  color="ts12"  storage="scene1.ks"  size="20"  x="260"  width="400"  y="240"  text="どちらかと言うと興味あり"  target="*selectinterest"  ]
[s]

------------------------------------------------------------------------------------------------
⑥コンフィグ画面
------------------------------------------------------------------------------------------------
　テーマに合ったコンフィグ画面を利用するためには
　[button]タグで sleepgame を指定してジャンプさせます。
　次のようなボタンでコンフィグ画面に移動して下さい。

　[button x=166 y=507 graphic="title/button_config.gif" storage="../others/plugin/theme_kopanda_12/config.ks" role="sleepgame"]

　storage がプラグイン以下の config.ks になっていないとエラーが発生します。

------------------------------------------------------------------------------------------------
⑦おまけ１・クリック待ちグリフ素材
------------------------------------------------------------------------------------------------
　テーマに合わせたクリック待ちグリフ素材を同梱しています。
　tyrano > images > system の中に入っている nextpage.gif に上書きしてご利用ください。

------------------------------------------------------------------------------------------------
⑦おまけ２・CGモード、回想モード素材
------------------------------------------------------------------------------------------------
　テーマプラグインと同じデザインでCGモードや回想モードを作成できる素材です
　導入方法につきましてはTYRANOS様の記事にて（https://tyranos.me/archives/686）
